# Running
## Setup
### CMAKE
```sh
mkdir build
cmake -S . -B build
```

### G++
```sh
g++ -std=c++20 -Wall src/*.cpp -leditline -o batlang
```

### Shell File
```sh
bash setup.sh
```

## Running
### CMAKE
```sh
cmake --build build
```

### G++
```sh
./batlang
```

>`run.sh` compile **and** run (using `cmake`)
